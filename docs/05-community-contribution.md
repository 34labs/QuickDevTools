# Community Contribution

DevTools Hub is designed to be a **community-driven project**.

Developers are encouraged to contribute new tools.

## Contribution Workflow

1. Fork the repository
2. Create a tool file
3. Register the tool
4. Add the tool to plugins/tools.json
5. Submit a pull request

## Tool File Location

Community tools should be placed in:

community-tools/

Each file represents one tool.

## Community Goals

The project aims to grow through:

- open contributions
- shared developer utilities
- hidden gem discoveries
