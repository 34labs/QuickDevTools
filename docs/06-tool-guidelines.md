# Tool Guidelines

To maintain quality, all tools must follow specific rules.

## Requirements

Every tool must:

- work offline
- use pure JavaScript
- avoid external APIs
- remain lightweight

## Performance

Tools should load quickly and avoid blocking the UI.

Recommended limits:

- under 200 lines of code
- minimal dependencies

## UI Consistency

Tools should:

- match existing design patterns
- support mobile devices
- provide clear results

Following these guidelines ensures the platform remains stable and efficient.
