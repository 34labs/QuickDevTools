# Project Roadmap

The roadmap outlines the future direction of DevTools Hub.

## Phase 1

Core developer tools:

- JSON formatter
- Base64 converter
- UUID generator
- Regex tester

## Phase 2

Productivity features:

- clipboard history
- snippet manager
- developer shortcuts
- focus timer

## Phase 3

Community ecosystem:

- plugin marketplace
- contributor leaderboard
- tool discovery system

## Long-Term Vision

DevTools Hub aims to become:

a community-powered developer productivity platform.

Similar inspiration includes:

- Raycast extensions
- VS Code extensions
- DevToys

But entirely browser-based.
